----Spirits DEMO - Zebra Soundset by Complex------------------------------------------------------

This is a 12 patches DEMO. If you like these sounds you can download the full soundset with 128 
patches from here: 

http://www.zensound.es/soundsets/zebra-spirits/

----Installation----------------------------------------------------------------------------------

Mac

Go to the folder 'Presets' and copy the folder 'ZenSound - Spirits DEMO' in this route. 
MacHD/Library/Audio/Presets/u-he/Zebra/

Windows

Go to the folder 'Presets' and copy the folder 'ZenSound - Spirits DEMO' in this route.
...\VstPlugins\Zebra.data\Presets\Zebra\
